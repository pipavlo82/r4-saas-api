HTTP API

Minimal, production-ready interface to RE4CTOR Core RNG and VRF node.

GET /v1/health

Health check

Lightweight liveness endpoint. Returns {"ok": true} when the gateway is online.

GET /v1/random

Raw randomness

Proxy to RE4CTOR Core RNG. Supports variable length output and multiple encodings.

GET /v1/vrf + POST /v1/verify

VRF + verification

Fetch dual-signed randomness and verify ECDSA recovery (ready for on-chain usage).

Endpoint Description Auth
GET /v1/health Simple health check ({"ok": true}). None
GET /v1/meta Gateway metadata (version, upstream URLs). None
GET /v1/random?n=16&fmt=hex Random bytes from RE4CTOR Core RNG (hex/base64/raw). X-API-Key or ?api_key=
GET /v1/vrf?sig=ecdsa Verifiable randomness with ECDSA + ML-DSA-65 signature. X-API-Key or ?api_key=
POST /v1/verify Recover signer from (msg_hash, v, r, s) and compare with expected address. X-API-Key (optional)
